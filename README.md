# pixel-towns
A small web project that generates random pixel art towns, each with a unique name. Built for fun and personal use.
Live site: pixeltowns.vercel.app
What it does
Every time you hit Generate, the site produces a new pixel art town alongside a randomly generated name for it. You can toggle background music using the gear icon.
How to use

Open the site
Click Згенерувати! / Generate to get a new town
Click the ⚙️ settings icon to turn the sound on/off
Enjoy your town
